Kimlik Doğrulama Hatası

GitHub kimlik doğrulama sırasında bir hata oluştu. Lütfen aşağıdaki talimatları izleyin.

GitHub OAuth Uygulaması Kurulumu:

  1. GitHub'da Developer Settings sayfasına gidin
  2. "OAuth Apps" bölümüne tıklayın ve "New OAuth App" seçin
  3. Gerekli bilgileri doldurun:
    • Application name: LibEdge CMS
    • Homepage URL:
    • Authorization callback URL:
  4. Oluşturduktan sonra Client ID ve Client Secret değerlerini alın
  5. Cloudflare Pages'de Environment Variables olarak ekleyin:
    • GITHUB_CLIENT_ID: [Client ID değeriniz]
    • GITHUB_CLIENT_SECRET: [Client Secret değeriniz]

Sorun Giderme Önerileri

  • GitHub OAuth uygulamasının doğru yapılandırıldığından emin olun
  • Callback URL'in doğru olduğundan emin olun
  • Cloudflare Pages environment variables'larını kontrol edin
  • Tarayıcı önbelleğini temizleyin (Ctrl+F5)
